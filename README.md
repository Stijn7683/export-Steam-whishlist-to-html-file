# export-Steam-whishlist-to-html-file
Using this chrome extention you can export any Steam whislist as an html file.

# how to use:
1. download the steam-whislist-exporter map with the files in it.
2. open the extentions tab in chrome
3. enable developer mode in the top right
4. click on load unpacked extention and select the folder you just downloaded
5. visit a Steam whishlist
6. click on the extention icon
7. click the button on the popup to download the Steam whislist curriently open
